# 🤖 Bot Detection System (Hackaton-2026-IT)

## 📌 О проекте

Проект разработан в рамках хакатона и решает задачу:

> **Выявление автоматизированной активности пользователей (ботов) на основе анализа данных**

Система использует методы машинного обучения для классификации пользователей на:

* 👤 **Human (реальный пользователь)**
* 👤🤖 **Probably bot (вероятно бот)**
* 🤖 **Bot (автоматизированный аккаунт)**

---

## 🧠 Как это работает

1. Собираются данные о поведении пользователей
2. Выделяются признаки (features)
3. Обучается модель машинного обучения
4. Модель предсказывает: бот или человек

Используется алгоритм:

> 🌳 **Random Forest Classifier**

## 🚀 Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/ariruaR/Hackaton-2026-IT.git
cd Hackaton-2026-IT
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate     # Linux / Mac
venv\Scripts\activate        # Windows
```

### 3. Установка зависимостей

(если нет requirements.txt — стоит добавить)

```bash

```

### 4. Запуск теста модели

```bash
python ml_test_claude.py
```

## 📊 Модель

Используется:

* **Random Forest Classifier**
* Обучена на поведенческих признаках пользователей

Файлы:
* `bot_detector_model.joblib` — модель
* `bot_detector_features.json` — список признаков
* `feature_importance.csv` — важность признаков
  
## 📈 Интерпретация модели

Файл `feature_importance.csv` показывает:
* какие признаки сильнее всего влияют на решение модели
* какие паттерны характерны для ботов

Это важно для:
* объяснимости модели
* улучшения качества
